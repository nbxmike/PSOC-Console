◆ Blinky_Task()

void Blinky_Task ( void *  arg)

Blinky_Task blinks the local heartbeat LED, defined here so there is always a starting task.

This task needs to be modified for each board that it is run on, for Cypress PSOC boards the following are the LED locations (incomplete I am aware): CY8KIT-059 - P2.1 CY8KIT-029 - P1.6 CY8KIT-040 - P3.1 and of course you need to map these pins to your TopDesign.cysch register names. Blink rate is presently hard coded for 1Hz.

◆ FRInit()

void FRInit ( )

FRInit initializes stuff needed for FreeRTOS.

FRInit initializes the interrupt hooks needed by FreeRTOS and also creates the LED heartbeat task just so there is always some task for FreeRTOS. The heartbeat can be deleted, I just like to see something happen when I start a project.
