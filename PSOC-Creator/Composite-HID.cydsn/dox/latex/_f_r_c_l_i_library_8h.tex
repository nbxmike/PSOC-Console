cli\+Task task which invokes the Free\+R\+T\+OS C\+LI engine 

cli\+Task process the user input on the command line. At present it only support character input and backspace editing; there is no support for cursor movement or anything else. To make things as platform independant as possible this accepts D\+EL or BS as backspace, accepts CR, LF, or C\+R-\/\+LF as input termination.
